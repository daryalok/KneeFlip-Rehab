# Screenshots

Official Flipper Application Catalog submissions need screenshots.

Screenshots should be captured with qFlipper's screenshot feature or an emulator/simulator if one is available. Do not resize, crop, decorate, or otherwise modify catalog screenshots.

Required screenshots:

1. Main menu
2. Ice timer
3. Quad sets timer
4. Session summary
5. About/disclaimer screen

Suggested placeholder filenames:

- `main-menu.png`
- `ice-timer.png`
- `quad-sets.png`
- `session-summary.png`
- `about.png`
